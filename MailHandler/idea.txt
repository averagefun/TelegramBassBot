Идея для рассылки:
Создаём таблицу очереди
ТЕКСТ | НОМЕР ПОСЛЕДНЕГО ПОЛЬЗОВАТЕЛЯ

Функция рассылки вызывается кроном каждый час (можно отрегулировать в зависимости от нагрузки)
и проверяет на очередь, дальше она берёт нового пользователя и отправляет ему.

CREATE TABLE mail_requests( msg_id INT PRIMARY KEY, last_user INT NOT NULL DEFAULT 0,
count INT NOT NULL DEFAULT 0, active BIT NOT NULL DEFAULT 0);
